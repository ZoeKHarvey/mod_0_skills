How to become a Turing student
1. Apply to Turing
  1.1 Fill out online application
    1.1.1Enter personal information
      1.1.1.1 Enter name
      1.1.1.2 Enter current location
    1.1.2 Upload current resume
      1.1.2.1 Update current job history
        1.1.2.1.1 Write down employer's name
        1.1.2.1.2 Include brief job description
        1.1.2.1.3 Include dates of employment
      1.1.2.2 Note education history
        1.1.2.2.1 Write level of highest accomplished education
          1.1.2.2.1.1Write name of institution
          1.1.2.2.3 Include dates of education
          1.1.2.2.4 Note main subject of study
      1.1.2.3 Include special skills
        1.1.2.3.1 Note any cerfifications
        1.1.2.3.2 Note personal strenghts
  1.2 Complete logic tests
    1.2.1 Prepare for logic tests
      1.2.1.1 Practice logic tests
        1.2.1.1.1 Find online logic test examples
        1.2.1.1.2 Complete online logic test examples
      1.2.1.2 Gather supplies
        1.2.1.2.1 Prepare computer
        1.2.1.2.2 Locate computer
          1.2.1.2.2.1Charge computer
          1.2.1.2.3 Check internet connection
        1.2.1.2.3 Prepare notebook
          1.2.1.2.3.1 Buy/find notebook
           1.2.1.2.3.1.1 Check to make sure there's enough blank pages
        1.2.1.2.4 Prepare writing materials
          1.2.1.2.4.1 Buy/find pen
          1.2.1.2.4.2 Buy/find pencil
    1.2.2 Complete logic tests
      1.2.2.1 Sign into timed logic test
        1.2.2.1.1 Sign into turing portal
      1.2.2.2 Start test
        1.2.2.2.1 Enter personal information
        1.2.2.2.2 Click "start"
      1.2.2.3 Read through questions
        1.2.2.3.1 Take notes in notebook
    1.2.3 Answer question on notebook
      1.2.3.1 Write down estimated answer
      1.2.3.2 Check multiple choice options
        1.2.3.2.1 Double check answer
          1.2.3.2.2 Go through solution again
      1.2.3.3 Answer the question
        1.2.3.3.1 Select which multiple choice answer fits your solution
    1.2.4 Submit Application
      1.2.4.1 Double-check information is correct
        1.2.4.1.1 Review personal information
          1.2.4.1.1.2Review spelling of name
        1.2.4.1.2 Review resume
          1.2.4.1.2.1 Spell check
          1.2.4.1.2.2 Check proper dates
      1.2.4.2 Press "Submit"

2. Complete an interview
  2.1 Set up interview online
    2.1.1 Go to Turing portal
      2.1.1.2 Navigate to "schedule and interview"
        2.1.1.2.1 Pick time slot for interview
      2.1.1.2 Check your schedule 
        2.1.1.2.1 Make note of picked time and date
  2.3 Answer interview questions
    2.3.1 Prepare for interview
      2.3.1.1 Complete mock interview questions
        2.3.1.2 Find mock interview questions online
          2.3.1.2.1 Google mock interview questions
    2.3.2 Test computer 
      2.3.2.1 Test audio
        2.3.2.1.1 Open Zoom
          2.3.2.1.2 Click "test audio"
      2.3.2.2 Test video
        2.3.2.2.1 Open Zoom
          2.3.2.2.1.1 Click "Text video"
    2.3.3 Open Zoom link to interview
      2.3.1 Open email
       2.3.1.1 Open email to confirmation of scheduled interview
       2. Click on Zoom link
